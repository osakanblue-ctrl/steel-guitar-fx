## 日本語

ブラウザで動くラップスティール／E9ペダルスティールギター・シンセサイザー。  
インストール不要。スマホ・タブレット・PCで試せます。

**▶ [ライブデモ](https://osakanblue-ctrl.github.io/steel-guitar-fx/)**

### 主な機能

- **Karplus-Strong合成**によるスティールギター音源
- **スライドバー**：リアルタイムピッチ・グライド（バー移動で音程が変化）
- **マルチタッチ対応**：左手でバー、右手で3フィンガーピッキング
- **LAP6 / E9ペダルスティール**切替
- **ペダルA/B**：踏んでいる間、対象弦がグライド
- **木材セレクター**（Maple / Mahogany / Koa / Rosewood / Walnut / Ash）
- **バー素材セレクター**（Steel / Glass / Brass / Ceramic）
- **ステップシーケンサー**：8〜32ステップ、ループ／ワンショット
- **パターン保存**：A〜Dスロット（localStorage）＋JSONエクスポート／インポート
- **MIDIエクスポート**：Format1、per-stringチャンネル、ピッチベンド対応
- **MIDIリアルタイム出力**対応

### 動作確認済み環境

- Android Chrome（マルチタッチ）
- Windows Chrome（マウス操作）

触ってみて気になった点をぜひ教えてください。

## English

A browser-based lap steel / E9 pedal steel guitar synthesizer.  
No installation required. Works on smartphone, tablet, and PC.

**▶ [Live Demo](https://osakanblue-ctrl.github.io/steel-guitar-fx/)**

### Features

- **Karplus-Strong synthesis** for steel guitar tone
- **Slide bar** with real-time pitch glide (pitch changes as the bar moves)
- **Multi-touch support**: bar hand + 3-finger picking hand simultaneously
- **LAP6 / E9 pedal steel** tuning modes
- **Pedal A/B**: target strings glide while pedal is held
- **Wood selector** (Maple / Mahogany / Koa / Rosewood / Walnut / Ash)
- **Bar material selector** (Steel / Glass / Brass / Ceramic)
- **Step sequencer**: 8–32 steps, loop / one-shot
- **Pattern save**: slots A–D (localStorage) + JSON export / import
- **MIDI export**: Format 1, per-string channels, pitch bend
- **MIDI real-time output** support

### Tested Environments

- Android Chrome (multi-touch)
- Windows Chrome (mouse)

- How playable it feels
- Anything that seems off

## Español

Sintetizador de guitarra steel (lap steel / E9 pedal steel) que funciona directo en el navegador.  
Sin instalación. Compatible con celular, tablet y PC.

**▶ [Demo en vivo](https://osakanblue-ctrl.github.io/steel-guitar-fx/)**

### Características

- **Síntesis Karplus-Strong** para sonido de guitarra steel
- **Barra deslizante** con glide de tono en tiempo real (el tono cambia al mover la barra)
- **Multi-touch**: una mano en la barra, la otra tocando las cuerdas con 3 dedos
- **Afinaciones LAP6 / E9 pedal steel**
- **Pedales A/B**: las cuerdas asignadas hacen glide mientras el pedal está presionado
- **Selector de madera** (Maple / Mahogany / Koa / Rosewood / Walnut / Ash)
- **Selector de material de barra** (Steel / Glass / Brass / Ceramic)
- **Secuenciador de pasos**: 8–32 pasos, loop / one-shot
- **Guardar patrones**: slots A–D (localStorage) + exportar / importar JSON
- **Exportar MIDI**: Formato 1, canales por cuerda, pitch bend
- **Salida MIDI en tiempo real**

### Entornos probados

- Android Chrome (multi-touch)
- Windows Chrome (ratón)
